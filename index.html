Permission control for Claude Code — from your menu bar
Approve/deny tool calls
Three modes: Present / Remote / Away
Workspace-aware auto-policies
The Problem

Claude Code asks for permission in the terminal. You context-switch, lose focus, approve blindly. When you step away, everything stalls — tools time out and work stops.

The Solution

A menu bar app that intercepts every permission request. Approve or deny in a floating popover. Switch to Away mode and Claude keeps working on safe operations inside your project.
